Domain Name

Domain names are easy to remember words that we can use to communicate to a DNS server the website you want to visit. The Domain Name System (DNS) is what translates the friendly name to an IP address.

The domain name hides the IP address that most people aren't interested in seeing or using, like the 172.217.23.14 address used by google.com.
